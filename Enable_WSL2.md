# How to enable WSL2 on Windows 11 Pro.

----------------------------------------------------------

Date: 14/10//2024

## Step 1: Enable WSL2

On Windows 11 Pro, WSL should be already enabled by default.

```
wsl --set-default-version 2
```
List all WSL linux distro online
```
wsl --list --online
```
List all WSL Linux distro installed on local PC.
```
wsl -l -v
```
